Automatic writing machine

Description: Designed and developed an innovative Automatic Writing Machine utilizing Arduino and CNC Shield technology. This project seamlessly blends hardware and software to enable precise and automated writing tasks. Incorporated CNC technology for intricate and customizable designs. The system showcases proficiency in Arduino programming, CNC machinery, and creative automation solutions. This project's applications range from artistic endeavors to industrial automation, showcasing adaptability and technical prowess.

Role: In the Automatic Writing Machine project, I played the crucial role of a software engineer. My responsibilities included utilizing Inkscape for intricate design work, creating detailed patterns, and incorporating text elements. As a software person, I played a pivotal role in translating these designs into machine-readable G-code instructions. Additionally, I managed the communication aspect using Universal G-code Sender (UGS), ensuring a seamless flow of instructions from the computer to the CNC machine. My role emphasized the integration of creative design with precise software execution, contributing significantly to the project's success.
